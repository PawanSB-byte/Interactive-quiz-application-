## 📌 Objective
To design and develop an interactive quiz application using **HTML, CSS, and JavaScript**.  
The quiz displays multiple-choice questions, accepts user responses, calculates the score, and shows the final result dynamically.

---


## 🛠️ Technologies Used
- HTML (Structure)
- CSS (Styling)
- JavaScript (Logic & Interactivity)

---

## ✨ Features
- Displays quiz introduction
- Shows multiple-choice questions
- Allows answer selection
- Next/Submit button navigation
- Score calculation
- Result message display (Excellent / Good / Try Again)
- Restart Quiz feature
- Student Name & Roll No displayed permanently on screen

---

## ⚙️ How to Run the Project

1. Download or clone the project folder.
2. Open the folder `interactive_quiz`.
3. Double-click `index.html`.
4. The quiz will open in your browser.
